# Front-end-rede-social
Desenvolva um website complete em HTML + CSS + JavaScript que, utilizando as operações do Webservice, seja capaz de realizar as operações esperadas da rede social, incluindo: Ler, criar, editar e deletar posts. Ler, criar, editar e deletar comentários. Dar e tirar likes. Cadastrar usuário. Fazer login de usuário.
